# Client Documentation Context

This folder contains documentation for the client to understand and manage the site.

## Files

- **setup-guide.md** - How to set up locally
- **product-management.md** - How to add/update products
- **deployment.md** - How to deploy updates
- **payment-setup.md** - PayFast/Yoco configuration
- **support.md** - Common issues and solutions

## Create These Files When Ready

Each file should be written for non-technical stakeholders:
- Clear, simple language
- Step-by-step instructions
- Screenshots where helpful
- Troubleshooting section

[Explain in plain English]
